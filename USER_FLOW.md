# SmartCart — User Flow Guide 🛒

## Overview
SmartCart (סל חכם) is a grocery basket price comparison app for Israeli supermarkets.
It lets you build a shopping list, compare prices across supermarkets, and find the cheapest option.

---

## Additional Features

### 🌙 Dark Mode
- Toggle via moon/sun icon in header
- Persists in localStorage
- Respects system preference on first visit

### 🌐 Language Toggle
- EN / עב button in header
- Switches all UI text between English and Hebrew
- Switches page direction (RTL ↔ LTR)

### 📤 Sharing & Export
- **Share basket**: Copies encoded URL to clipboard
- **Import shared basket**: Open URL with `?share=` param
- **Export comparison**: Copy to clipboard, WhatsApp, or download .txt

### 💰 Price Drop Notifications
- Green banner on basket page when item prices have decreased
- Shows product name, supermarket, old → new price, % drop
- Dismissible per session

### 💾 Saved Baskets
- Save current basket with a custom name
- Load any saved basket (up to 10)
- Stored in cookies — works without login

### 📱 PWA
- Installable on mobile via "Add to Home Screen"
- Offline shell caching via service worker
- Custom app icons and manifest
